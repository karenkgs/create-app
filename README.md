#### Custom boilerplate code generator.

### Setup
##### Dependencies for running this project
```sh
$ npm
```

##### Install project
Inside of the root folder
```sh
$ npm install generate-app
```

##### Usage
```sh
$ generate-app
```

##### Run locally
Inside of the root folder
```sh
$ npm start
```
